## 📌 About Project

Ajay's AI Marketing Lab is an AI-powered platform that helps users create marketing strategies.

It provides assistance for:

- SEO
- Ads
- Content Strategy
- Blog Ideas
- Email Campaigns


## ✨ Features

✅ AI Marketing Assistant

✅ OpenAI + Gemini Integration

✅ Automatic AI fallback system

✅ User Authentication

✅ Protected Dashboard

✅ SEO Generator

✅ Ads Generator

✅ Blog Generator

✅ Email Generator
